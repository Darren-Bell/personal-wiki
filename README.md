# Darren's Knowledge Base

Personal and work knowledge base built on the [Portent](https://portent.md) spec.

## Types

- [[project]] — bounded efforts with start/end
- [[operation]] — recurring work, completable in one sitting
- [[responsibility]] — long-running areas of accountability
- [[task]] — one-off items (prefer external task manager)
- [[event]] — things that happened
- [[note]] — durable knowledge artifacts
- [[topic]] — areas of interest
- [[person]] — people and agents

## Responsibilities

- [[manage-family-and-household]]
- [[stay-organized]]
- [[manage-email]]

## How to Use

1. **Capture** — save thoughts, links, notes quickly. Don't worry about structure.
2. **Organize** — assign type + relationships. Mark `_organized: true`.
3. **Archive** — when done, mark `archived: true`.
